# Contributing

Development conventions for this repository are documented in [CLAUDE.md](CLAUDE.md).

## Quick start

```sh
make setup   # install dependencies and set up pre-commit hook
make build   # build all packages
make test    # run all tests
make lint    # run biome linter
```

## Commit style

[Conventional Commits](https://www.conventionalcommits.org/) in English. One commit per logical unit of change. Work on branches and merge via pull request — direct commits to `main` are not permitted.

## Workflow

Issue registration and roadmap updates follow the `pfd-ops` skill protocol. See [.claude/skills/pfdsl/](.claude/skills/pfdsl/) for the installed skill.
